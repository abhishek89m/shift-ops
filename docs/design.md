# Design

Compact visual system note for `Shift Ops`.

## Direction

- calm
- operational
- mobile-first
- rounded, low-chrome surfaces
- warm coral accent on warm off-white background
- near-black text for outdoor readability

## Layout rules

- top progress block stays compact
- only one obvious next action at a time
- cards over tables
- mobile opens detail as its own step
- tablet/desktop keep list + detail split

## UI rules

- avoid hero-marketing copy
- keep labels short
- show task ids
- keep status sections explicit:
  - in progress
  - remaining
  - done or skipped
