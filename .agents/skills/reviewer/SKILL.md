# Reviewer

Use this skill when a change needs a fast product and code review.

## Review Against

- current commit scope
- product fit for a field worker
- clarity of implementation
- interview explainability
- dependency weight

## Prioritize Findings

1. scope drift
2. product mismatch
3. broken verification story
4. unnecessary abstraction

## Good Outcomes

- one clear story
- working checks
- small diff
- easy to justify in interview
